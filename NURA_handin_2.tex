\section{Satellite galaxies around a massive central}

\subsection{(1a) Determining the normalization}

\textbf{Explain the integration method used (e.g. Romberg integration or another Richardson method).  
Describe how you implemented the spherical volume element \(dV\).}
The satellite galaxies around a central follow the number density
\begin{equation}
    n(x) = A\langle N_{sat}\rangle\left(\frac{x}{b}\right)^{a-3}\exp\left[-\left(\frac{x}{b}\right)^c\right]
\end{equation}

where $x$ is the relative radius to the virial radius $(x\equiv r/r_\mathrm{vir})$. This distribution has to normalize to $\langle N_{sat}\rangle$ when integrating from $x_\mathrm{min}=0$ to $x_\mathrm{max}=5$. 

\begin{equation}
    \int dV n(x) = \langle N_{sat}\rangle
\end{equation}

Rewriting $dV$ in spherical coordinates, we find that $dV = 4\pi x^2$. Therefore, the integral that determines the value for $A$ is

\begin{equation}
    \int_{x=0}^{x=5} dx\; 4\pi x^2n(x) = \langle N_{sat}\rangle
\end{equation}

Filling in the equation for $n(x)$ and isolating $A$ results in
\begin{equation}\label{eq:normalization constant}
    A = \frac{\langle N_{sat}\rangle}{\int_{x=0}^{x=5} dx\; 4\pi x^2 \langle N_{sat}\rangle\left(\frac{x}{b}\right)^{a-3}\exp\left[-\left(\frac{x}{b}\right)^c\right]}
\end{equation}

To evaluate this integral, we will use Romberg integration. This numerical integration technique combines different estimates of the integral $\int f(x)dx$ into a better approximation. It starts by finding the area between the two endpoints of the integral $r_0$. For the next estimate, it uses the previous estimate and the sum of function evaluations between the endpoints, scaled by the step size $r_{i+1}=\frac{1}{2}\left[r_i+\Delta_{i+1}\sum_j f (x_j)\right]$. 

After repeating this step $m-1$ times, there are $m$ estimates for the value of the integral. By using the equation 
\begin{equation}
    r_{i,j+1}=\frac{4^{j+1}{r_{i+1, j}-S_{i,j}}}{4^{j+1}-1}
\end{equation}
where the initial values are $r_{i,0}$. This step repeats until there is a single value remaining that uses a combination of all the starting estimates. 

Applying the Romberg integration to \autoref{eq:normalization constant}, we find the following value for the normalization constant.

\subsection{(1b) Sampling the satellite distribution}

\textbf{Describe the sampling method used (e.g. rejection sampling).  
Explain how the random number generator was implemented and how its randomness was tested.}
We would like to sample random galaxies from the number density. To find the probability density, we need to find the number of galaxies between $x$ and $x+dx$. 
\begin{equation}
    \int dx\;N(x) = \int dx\;4\pi x^2n(x)
\end{equation}
So we get $N(x)dx=4\pi x^2n(x)dx$. The probability density needs to be normalized so we get
\begin{equation}
    p(x)dx = \frac{N(x)dx}{N_\mathrm{sat}} = \frac{4\pi x^2n(x)dx}{N_\mathrm{sat}}
\end{equation}
To sample from this distribution, we will use rejection sampling. This is most efficient when the \textbf{MORE EXPLANATION}.
First, a random x value $x_\mathrm{rand}$ is sampled from a uniform distribution from 0 to 5. Next, a random y value $y_\mathrm{rand}$ is sampled from a uniform distribution from 0 to 1. If $p(x_\mathrm{rand}) > y_\mathrm{rand}$, the $x_\mathrm{rand}$ is accepted into the sample. Otherwise, a new x and y value are sampled until the required number of samples is reached.

\begin{figure}[H]
\centering
\includegraphics[width=0.85\textwidth]{\PlotDir/my_solution_1b.png}
\caption{Comparison between the analytical satellite distribution \(N(x)\) and the histogram of 10 000 sampled satellite galaxies in logarithmic bins.}
\end{figure}

The sampled distribution largely agrees with the analytical prediction. However, below $\sim 10^{-2}$ there are no samples, whereas the analytical prediction is non-zero. This is due to the range of the relative radius values. There is a small probability that a uniformly sampled x value falls inside this range. Additionally, the probability density in this range is also small, such that even when a uniformly sampled x value falls inside this range, the change that is accepted into the sample is also small.

\subsection{(1c) Selecting and sorting satellites}

\textbf{Describe the selection of 100 galaxies (equal probability, no duplicates, no rejection).
Explain the sorting algorithm you used.}
We would like to make a selection of 100 galaxies from our sample of 10000 galaxies. For this, we want to shuffle the galaxies randomly and take the first 100. This is done by creating an array with 10000 random floats and an array with elements 0, 1, 2, ...,  10000-1. Next, the array with random numbers is sorted by swapping different elements. The same swaps are performed on the index array. Once the array with random numbers is sorted, we will have obtained an index array that is randomized. The first 100 random indices are used to make the selection of galaxies. The relative radii are then sorted to find the number of galaxies within different relative radii.
